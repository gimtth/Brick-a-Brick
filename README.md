# 砖了个砖 - 自动化辅助脚本

针对"砖了个砖"游戏的自动化解决方案，基于OpenCV图像识别和GBFS贪心算法。

## 功能特性

- **智能求解**：使用GBFS贪心算法，保证找到解决方案
- **图像识别**：基于OpenCV的多尺度模板匹配，自动识别游戏方块
- **可视化预览**：求解后可视化展示每一步，无需执行即可验证
- **暂停/继续**：支持暂停执行，处理异常情况后继续

## 安装

```bash
pip install -r requirements.txt
```

## 使用方法

```bash
python game_auto_player.py
```

### 操作流程

1. **选择游戏区域**：首次运行时，按提示用鼠标框选游戏区域
2. **选择模式**：
   - 模式1：GBFS智能求解（推荐，通关率非常高）
   - 模式2：逐轮自动游戏（通关率非常低）
3. **预览方案**：查看可视化步骤，确认是否执行
4. **执行/取消**：确认后自动执行，或取消

### 快捷键

| 操作 | 快捷键 |
|------|--------|
| 暂停 | 鼠标移到屏幕左上角 |
| 继续 | 鼠标移回游戏区域 |
| 退出 | 暂停时按 `q` 键 |

## 技术栈

- **算法**：GBFS（贪婪最佳优先搜索）- 贪心算法
- **图像识别**：OpenCV (cv2) - 多尺度模板匹配
- **GUI**：Tkinter
- **自动化**：pyautogui, keyboard

## 算法说明

### GBFS求解（贪心算法）

- **算法类型**：贪婪最佳优先搜索（Greedy Best-First Search）
- **状态空间**：所有可能的棋盘配置
- **启发函数**：可消除方块对的数量
- **贪心策略**：每步选择启发值最优的状态
- **保证**：一定能找到解决方案
- **步数**：固定为 `方块总数 / 2` 步

### 图像识别（OpenCV）

- **框架**：OpenCV (cv2)
- **方法**：多尺度模板匹配（5个尺度）
- **坐标聚类**：确定网格结构
- **自动过滤**：去除无效识别
